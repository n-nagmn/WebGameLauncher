# Web Game Launcher

モダンで洗練されたUIデザインを採用した、高機能ゲームランチャーです。
自作のWebゲームや、頻繁にアクセスするブラウザゲームを1つのダッシュボードでスタイリッシュに管理できます。PHPとJSONを用いたデータ保存により、複数端末からのアクセス・データ共有に対応しています。

## 主な機能

- **直感的な UI/UX**: ダークテーマを基調とした、シンプルで没入感のある操作感。
- **サーバー連携によるデータ共有**: 追加したゲームリストやシステム設定は、PHPを介してサーバー上のJSONファイルに自動保存されます。どの端末からアクセスしても同じ環境を維持できます。
- **AIイメージ生成 (擬似)**: サムネイル未指定時、タイトルからハッシュ値を計算。Canvas APIを用いて、タイトル固有の配色とイニシャルを持つサムネイルを自動生成します。
- **インタラクティブな操作**:
  - **ドラッグ&ドロップ**: カードを自由に並び替えて優先順位を変更。
  - **リアルタイム検索**: 増えすぎたゲームをタイトル名で即座にフィルタリング。
- **高度なカスタマイズ**:
  - カードサイズ（小・中・大）、角丸の半径、カード間の余白調整。
  - アクセントカラー（テーマ色）の変更。
  - カスタム背景画像（壁紙）の設定。
  - 時計の12h/24h表記切り替え、新規タブ展開の制御。
- **堅牢なデータ管理**: JSON形式でのエクスポート/インポート機能によるバックアップと移行。

## 技術スタック

- **Language**: JavaScript (ES6+), HTML5, CSS3, PHP
- **Architecture**: Vanilla JS (No Frameworks)
- **Graphic**: Canvas API (Dynamic Thumbnail Generation)
- **Storage**: JSON files via PHP (Server-side storage)

## 導入方法

1.  PHPが動作するWebサーバー環境（Apache, Nginx等）を用意します。
2.  以下のファイルをサーバーの同一ディレクトリに配置します：
    - `index.html`
    - `save_config.php`
    - `save_games.php`
    - `config.json`
    - `games.json`
3.  サーバーがJSONファイルに書き込めるよう、`config.json` と `games.json` のパーミッションを書き込み可能（`666` など）に設定してください。
4.  任意のモダンブラウザ（Chrome, Edge, Firefox, Safari等）で `index.html` にアクセスします。
5.  画面右上の「⚙️（設定）」または「＋（追加）」から、あなたのゲームを登録して開始してください。

## システム構成とカスタマイズ

### ゲームの追加 (Game Editor)
- **タイトル**: 検索対象および自動生成画像のソース文字となります。
- **URL**: `https://` から始まるゲームのリンク先を指定します。
- **サムネイル**: 外部URLを指定可能。空欄時はシステムが配色を自動決定します。

### 詳細設定 (System Config)
設定モーダルから以下の項目を永続的に変更可能です（設定内容は `config.json` に保存されます）。

| カテゴリ | 項目 | 説明 |
| :--- | :--- | :--- |
| **表示** | カードサイズ | 画面密度に合わせて3段階から選択 |
| | 時計形式 | 12時間/24時間表示の切り替え |
| **外観** | アクセントカラー | システム全体の強調色をカラーピッカーで指定 |
| | 壁紙 | 外部画像のURLを指定して背景を変更 |
| | 形状 | 角丸の半径やカード間のギャップを調整 |
| **動作** | ターゲット | 同一タブか新規タブかの挙動を選択 |
| | アニメーション | ホバー時の拡大エフェクトの有無 |

### データ管理 (Data Management)
設定画面内の「データ管理」セクションを利用します。
- **エクスポート**: 現在の全ゲームリストと設定を `launcher_backup_YYYY-MM-DD.json` として保存します。
- **インポート**: 保存したJSONファイルを読み込み、サーバー側のデータを上書き復元します。
- **初期化**: サーバー上の全てのゲームデータを削除し、クリーンな状態に戻します。
