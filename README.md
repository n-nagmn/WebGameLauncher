# Web Game Launcher

モダンで洗練されたUIデザインを採用した、高機能ゲームランチャーです。
自作のWebゲームや、頻繁にアクセスするブラウザゲームを1つのダッシュボードでスタイリッシュに管理できます。PHPとJSONを用いたデータ保存により、複数端末からのアクセス・データ共有に対応しています。

## 主な機能

### ゲーム管理
- **直感的な UI/UX**: ダークテーマを基調とした、シンプルで没入感のある操作感。
- **サーバー連携によるデータ共有**: 追加したゲームリストやシステム設定は、PHPを介してサーバー上のJSONファイルに自動保存されます。どの端末からアクセスしても同じ環境を維持できます。
- **AIイメージ生成 (擬似)**: サムネイル未指定時、タイトルからハッシュ値を計算。Canvas APIを用いて、タイトル固有の配色とイニシャルを持つサムネイルを自動生成します。
- **カテゴリ分類**: アクション、パズル、RPG、シューティング、スポーツ、ストラテジー、カジュアル、レトロの8カテゴリで整理。
- **プレイ人数管理**: 最小～最大の範囲指定でプレイ人数を記録。サムネイル右下にバッジ表示。

### インタラクティブな操作
- **ドラッグ&ドロップ**: カードを自由に並び替えて優先順位を変更（検索中は不可）。
- **リアルタイム検索**: 増えすぎたゲームをタイトル名で即座にフィルタリング。
- **キーボードナビゲーション**: `↑↓←→` でカード移動、`Enter` で起動、`N` で追加、`/` で検索、`Del` で削除。
- **フィルタリング**: カテゴリ別・プレイ人数別でゲームを絞り込み。

### 高度なカスタマイズ
- **カードサイズ**: 小(180px)・中(240px)・大(300px)・特大(360px)から選択。
- **角丸・間隔**: カードの角丸半径(0～24px)とカード間の余白(10～48px)を調整。
- **アクセントカラー**: システム全体の強調色をカラーピッカーで自由に指定。
- **壁紙**: 外部画像URLを指定して背景を変更。
- **時計**: 12h/24h表記切り替え。
- **タブ動作**: 同一タブまたは新規タブでゲームを開く。
- **アニメーション**: ホバー時の拡大エフェクトON/OFF。

### 堅牢なデータ管理
- **自動バックアップ**: 保存時にサーバー側で自動バックアップを作成（最新5世代保持）。
- **ローカルバックアップ**: ブラウザのlocalStorageに自動バックアップ。サーバー接続失敗時に復元可能。
- **エクスポート/インポート**: JSON形式でデータをバックアップ・移行。
- **レート制限**: IPベースのレート制限で過負荷を防止。
- **データ検証**: サーバー側でJSONスキーマ・型・サイズを厳密に検証。

## 技術スタック

- **Language**: JavaScript (ES6+), HTML5, CSS3, PHP 7.4+
- **Architecture**: Vanilla JS (No Frameworks), Module Pattern (IIFE)
- **Graphic**: Canvas API (Dynamic Thumbnail Generation)
- **Storage**: JSON files via PHP (Server-side) + localStorage (Client-side backup)
- **Security**: XSS escape, Rate limiting, Input validation, CORS headers

## 導入方法

1. PHPが動作するWebサーバー環境（Apache, Nginx等）を用意します。
2. 以下のファイルをサーバーの同一ディレクトリに配置します：
   - `index.html`
   - `save_config.php`
   - `save_games.php`
   - `config.json`
   - `games.json`
3. サーバーがJSONファイルに書き込めるよう、以下のパーミッションを設定してください：
   ```bash
   chmod 666 config.json games.json
   chmod 755 .
   ```
4. 任意のモダンブラウザ（Chrome, Edge, Firefox, Safari等）で `index.html` にアクセスします。
5. 画面右上の「⚙️（設定）」または「＋（追加）」から、あなたのゲームを登録して開始してください。

> **注意**: SELinux/AppArmor環境では追加のコンテキスト設定が必要な場合があります。

## システム構成とカスタマイズ

### ゲームの追加・編集 (Game Editor)
カード右上の⚙️ボタン、または「＋」カードから編集モーダルを開きます。

| 項目 | 必須 | 説明 |
| :--- | :--- | :--- |
| **タイトル** | ✅ | 検索対象および自動生成画像のソース文字 |
| **カテゴリ** | - | 8種類から選択。ヘッダーでフィルタリング可能 |
| **プレイ人数** | - | 最小～最大で範囲指定。サムネイル右下に👤/👥バッジ表示 |
| **ゲームURL** | ✅ | `https://` から始まるゲームのリンク先 |
| **サムネイル** | - | 外部URLを指定。空欄時は自動生成 |

### 詳細設定 (System Config)
設定モーダルから以下の項目を永続的に変更可能です（`config.json` に保存）。

| カテゴリ | 項目 | 選択肢/形式 | 説明 |
| :--- | :--- | :--- | :--- |
| **表示** | カードサイズ | 180/240/300/360px | 画面密度に合わせて選択 |
| | 時計形式 | 12h / 24h | 表示形式の切り替え |
| | タイトル表示 | ON/OFF | カード下部のタイトルバー表示制御 |
| **外観** | アクセントカラー | #RRGGBB | システム全体の強調色 |
| | 壁紙 | URL | 背景画像をカスタマイズ |
| | 角丸 | 0/4/8/16/24px | カードの角丸半径 |
| | カード間隔 | 10/20/32/48px | グリッドギャップ |
| **動作** | 新規タブ | ON/OFF | ゲームリンクの開き方 |
| | アニメーション | ON/OFF | ホバー時の拡大エフェクト |
| | ローカルバックアップ | ON/OFF | ブラウザに自動バックアップを作成 |
| **データ** | エクスポート | ダウンロード | 全データをJSONで保存 |
| | インポート | ファイル選択 | JSONファイルで復元 |
| | 初期化 | 確認後実行 | 全ゲームデータを削除 |

### キーボードショートカット

| キー | 動作 |
| :--- | :--- |
| `↑↓←→` | カード間を移動 |
| `Enter` | 選択中のゲームを起動 |
| `N` | 新規ゲーム追加モーダルを開く |
| `S` | 設定モーダルを開く |
| `/` | 検索ボックスにフォーカス |
| `Del` | 選択中のゲームを削除 |
| `Esc` | モーダルを閉じる / 検索・フィルターをリセット |

### フィルタリング

ヘッダー左部のチップボタンでゲームを絞り込みます。

- **カテゴリフィルター**: すべて / アクション / RPG / パズル ...
- **プレイ人数フィルター**: 全人数 / 1人 / 1～2人 / 1～4人 ...
- **範囲マッチング**: プレイ人数フィルターは範囲の重なりで判定（例: `1～4人` フィルターは `2～2人`、`3～8人` のゲームも表示）

### データ管理 (Data Management)

設定画面内の「データ管理」セクションを利用します。

- **エクスポート**: ゲームリスト・設定・タイムスタンプを含むJSONを `launcher_backup_YYYY-MM-DD.json` としてダウンロード。
- **インポート**: バックアップJSONを読み込み、サーバー側のデータを上書き復元。新旧フォーマット両対応。
- **初期化**: 全ゲームデータを削除。確認ダイアログ付き。

### 自動バックアップ仕様

サーバー側で以下の自動バックアップが動作します：

- **保存時バックアップ**: `games.json` / `config.json` 保存時に `backups/` ディレクトリへコピー
- **世代管理**: 最新5ファイルを保持、古いものは自動削除
- **復元**: バックアップファイルは手動で `games.json` / `config.json` へリネームして復元可能

## トラブルシューティング

| 症状 | 原因 | 対処 |
| :--- | :--- | :--- |
| 「サーバー保存に失敗」 | ディレクトリ書き込み権限不足 | `chmod 666 games.json config.json; chmod 755 .` |
| 「バックアップディレクトリ作成に失敗」 | PHPプロセスの権限制限 | `mkdir -p backups; chmod 777 backups` |
| データが復元されない | サーバー接続エラー | ローカルバックアップ設定をONにし、ページを再読み込み |
| 画像が表示されない | URLが無効またはCORS制限 | 画像URLを確認、または空欄で自動生成を利用 |
| カードが並び替えできない | 検索またはフィルター中 | 検索をクリアし「すべて」「全人数」を選択 |
